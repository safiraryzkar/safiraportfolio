Selected Projects

Classifier Breast Cancer Wiconsin in sklearn

Analysis of breast cancer Wisconsin using python programming language, the dataset was prepocessed using encoding and applying the min-max scaler. After preprocessing, the data was partitioned using cross-validation and classification models were created using logistic regression, decision tree, and naive Bayes algorithms. The performance of each classifier was evaluated using confusion matrices. The results revealed that both logistic regression and naive Bayes models achieved the highest accuracy of 90% in determining whether a patient has cancer or not.

Analysis of 'Pinjol' Tweets through Twitter Data Scraping and Advanced Text Processing Techniques

This study employs Twitter data scraping to analyze 'Pinjol' tweets, utilizing Sastrawi for linguistic preprocessing, which involves stemming and preprocessing for removing stop word and link. The analysis includes word cloud generation for preprocessing validation, TF-IDF for K-means clustering to group themes in the collected tweets, and social network analysis with centrality measures to evaluate user influence. This comprehensive approach aims to reveal patterns within the 'Pinjol' discourse on Twitter.
